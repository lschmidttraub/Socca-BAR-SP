# Snippet Registry

Maps local snippet folders under `snippets/` to their GitLab snippet IDs
in the [UCL 2025-26 snippet repository](https://gitlab.ethz.ch/socceranalytics/uefa-cl-2025-2026/-/snippets).
Reference snippets from the wiki via `$<id>` (e.g. `$2765`).

| Local folder                                       | Snippet ID | Description                                                      |
|----------------------------------------------------|------------|------------------------------------------------------------------|
| [`set_piece_statistics/`](set_piece_statistics/)   | $2765      | Offensive / defensive / per-match set-piece statistics           |
| [`player_physicality/`](player_physicality/)       | $2769         | Top-6 outfield-player mean-height comparison  |
